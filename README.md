## Project Name
AthleteHub – Integrated Athlete Management System

## Project Description
AthleteHub is a database-driven web application with an integrated platform designed to streamline athlete management, communication, performance tracking, and development for student-athletes and professional athletes.

Our system centralizes athlete data, team scheduling, performance statistics, medical records, and communication into one secure relational database system with role-based access control.
